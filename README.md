# USELF Website 🌍

This repository is dedicated to holding our **USELF Website Finals project** — a web project created to demonstrate frontend, backend, and research skills through a purpose‑driven website. The project also highlights climate awareness and related themes.

---

## 🧠 What Is the Purpose?

The goals of this project are:

- To **showcase our Frontend & Backend development skills**
- To demonstrate **research and design abilities**
- To remind the world that **the future of the planet matters**  
  — and to visualize this through a meaningful website experience

---
